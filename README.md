## 🧠 The Psychology Behind Sturdy

### **Attachment Theory Foundation**

Every script follows the **"Two Things Are True"** framework:
1. **Validate** the child's feeling (attachment secure)
2. **Set boundary** on behavior (safety first)

Example:
> *"I see you're angry. AND I won't let you hit me. You can punch a pillow."*

This combines Dr. Becky Kennedy's Good Inside framework with modern attachment-based parenting.

### **IFS (Internal Family Systems) Integration**

Scripts help you recognize your child's **parts** (not diagnose):
- The "frozen" part (shutdown during overwhelm)
- The "protective" part (aggression as safety)
- The "bored" part (screen addiction as stimulation)

By naming the part, you bypass shame and access compassion.

### **Neurotype Customization**

We customize scripts for:
- **ADHD kids**: Emphasize novelty, movement, breaks
- **Autistic kids**: Sensory-aware, predictable, clear boundaries
- **Anxious kids**: Reassurance, connection, control
- **PDA-profile kids**: Collaborative, low-demand, autonomy

## 🛡️ Safety & Privacy

### **What We Don't Do**

- ❌ Diagnose conditions (we suggest patterns, not labels)
- ❌ Replace therapy (we refer to licensed professionals)
- ❌ Offer medical advice (we stay in coaching lane)
- ❌ Train AI on your child's data (zero data retention from OpenAI)

### **What We Do**

✅ **Hard Safety Guardrails**
- Keywords like "self-harm" or "suicidal" → instant crisis hotline (988, Crisis Text Line)
- No medical/medication language in outputs
- Human-vetted scripts for high-risk situations
- Clear liability disclaimers in every generation

✅ **Privacy-First Architecture**
- PII scrubbed before sending to OpenAI (child name → [CHILD])
- Encrypted database columns for sensitive data
- No third-party tracking or ads
- Optional usage analytics (you control what's shared)

✅ **Transparent Terms**
- No hidden charges (what you see is what you pay)
- 1-click cancellation (instant, no gatekeeping)
- 30-day money-back guarantee on paid plans
- COPPA-compliant (child data treated with strictest care)

**Tech Stack:**

- **Backend**: Supabase (PostgreSQL + Auth + Real-time)
- **AI**: OpenAI GPT-4o-mini (with safety guards)
- **Payments**: Stripe (Weekly/Monthly/Lifetime subscriptions)
- **Deployment**: EAS Build for iOS/Android *(Note: NOT Vercel - see [VERCEL_MIGRATION.md](../VERCEL_MIGRATION.md))*
- **Database**: PostgreSQL with Row Level Security (RLS)

**Setup:**

```bash
# Clone repo
git clone https://github.com/yourusername/sturdy.git
cd sturdy

# Install root dependencies
npm install

# Install mobile app dependencies
cd sturdy-app
npm install

# Set up environment
cp .env.example .env
# Add: EXPO_PUBLIC_SUPABASE_URL, EXPO_PUBLIC_SUPABASE_ANON_KEY

# Run mobile app
npm start

# Choose platform:
# - Press 'i' for iOS simulator (Mac only)
# - Press 'a' for Android emulator
# - Press 'w' for web preview (testing only)
# - Scan QR code with Expo Go app for physical device
```

**Database Setup:**

```sql
-- Run in Supabase SQL Editor
-- Users (Supabase Auth handles this)

-- Profiles
CREATE TABLE profiles (
  id uuid PRIMARY KEY,
  email text UNIQUE,
  subscription_tier text DEFAULT 'free',
  created_at timestamp DEFAULT now()
);

-- Children
CREATE TABLE children (
  id uuid PRIMARY KEY,
  parent_id uuid REFERENCES profiles(id),
  name text,
  birth_date date,
  neurotype text,
  created_at timestamp DEFAULT now()
);

-- Scripts (generated)
CREATE TABLE scripts (
  id uuid PRIMARY KEY,
  parent_id uuid REFERENCES profiles(id),
  child_id uuid REFERENCES children(id),
  situation text,
  script_text text,
  validation text,
  reframe text,
  insight text,
  user_rating int,
  created_at timestamp DEFAULT now()
);
```

## ❓ FAQ

**Q: Is this therapy?**
A: No. Sturdy is an educational coaching tool. If your child needs mental health support, we refer you to licensed professionals.

**Q: What if the script doesn't work?**
A: You rate it, we adjust. After 3+ uses, our system learns what works for your child and future suggestions reflect that pattern.

**Q: Can I share scripts with my co-parent?**
A: Yes (coming in Phase 2). You invite your partner, they see your scripts, and the app suggests shared language.

**Q: What about my privacy?**
A: We never train AI on your data. Child names are scrubbed before sending to OpenAI. We don't sell data. We don't track you. Period.

**Q: Do I lose access if I cancel?**
A: Monthly plans: Yes, you lose journal access. Lifetime: Your data is yours forever, even if you cancel (won't happen—it's lifetime 😉).

**Q: Does this work for my neurotype of child?**
A: We optimize for ADHD, autism, PDA, anxiety. If your child has a rare neurotype, you can describe their patterns and the system adapts.
