# Sturdy-App Repository

This repository contains two separate applications:

## 📱 Sturdy App (Expo/React Native)
Located in `/sturdy-app`

A cross-platform mobile application built with Expo and React Native.

### Running the Expo app:
```bash
cd sturdy-app
npm install
npm start
```

## 🌐 Next.js Web App
Located in `/nextjs-app`

A web application built with Next.js.

### Running the Next.js app:
```bash
cd nextjs-app
npm install
npm run dev
```

## Deployment
- **Expo App**: Deployed to Vercel (configured via `vercel.json`)
- **Next.js App**: Can be deployed separately to Vercel or another platform
