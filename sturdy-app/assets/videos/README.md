# Video Assets

## Onboarding Video

Place your onboarding video here as `onboarding.mp4`.

### Video Specifications

- **Format**: MP4 (H.264 codec)
- **Resolution**: 1080p or 720p recommended
- **Frame Rate**: 30fps
- **File Size**: < 10MB (optimize for mobile performance)
- **Duration**: 10-30 seconds (will loop automatically)

### How to Add Your Video

1. Export or download your video in MP4 format
2. Optimize it for mobile (compress if needed)
3. Rename it to `onboarding.mp4`
4. Place it in this directory (`sturdy-app/assets/videos/`)
5. Restart your Expo dev server

### Fallback Behavior

If the video file is missing or fails to load:
- The app will show a gradient background instead
- All functionality will work normally
- An error message will be logged to the console

### Example Sources

You can create or source videos from:
- Stock video sites (Pexels, Unsplash, Pixabay)
- Custom recorded footage
- AI-generated backgrounds
- Animated gradients

Make sure you have proper licensing for any video you use.
